Основы CSS

Что такое CSS?

CSS (Cascading Style Sheets) — это язык описания стилей, используемый для оформления веб-страниц. Он позволяет задавать внешний вид элементов HTML, включая цвет, шрифты, отступы, размеры и расположение на странице.

Основные свойства CSS

  • color — задаёт цвет текста.
  • background-color — устанавливает цвет фона элемента.
  • font-size — изменяет размер текста.
  • margin — определяет внешние отступы вокруг элемента.
  • padding — задаёт внутренние отступы внутри элемента.

Пример использования CSS

CSS используется совместно с HTML для создания красивых и функциональных веб-страниц. Например, вот простой код:

body {
    background-color: lightblue;
    color: black;
}
h1 {
    text-align: center;
}
p {
    font-size: 16px;
}
